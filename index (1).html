This will also allow us to share our fiels as well because we are using github. This will allow employers to view our work if they googled our github.

Highlight #2 History of Git

Git began with a lot of controversy and was made because of necessity. With the break down between Linux kernel and Bitkeep. This prompted those in the Linux community to develop their own tools, which turned out better in the end. When they made this new system they had a few goals in mind. They used a simple design to create a fast and fully distributed platform. From using it in class, I was very surprised as to how easy it was to use. I honestly thought that when I was first introduced to it that I would have more problems with it then I did.

Highlight #3 Git Basics

Git stores and uses information much differently then other systems may. Git thinks about its data in a different way. Most systems store information in a list. Git doesn’t store data in this way. Git thinks of information more as a picture. It takes a snap shot of what you save. So this way git uses data as a stream of snapshots. This makes it a much better to use, as its tools are far more useful. All the time I have spent using git for this class has more then provide its usefulness. The presentation we had in class about git was very true about the importance of it.
